
# Sistema de Análise Financeira com IA

## Descrição
Sistema web para automatizar a extração e análise de dados financeiros a partir de documentos
como balanços patrimoniais, DREs e planilhas contábeis.

## Objetivo
Reduzir o tempo de análise manual de documentos financeiros e gerar indicadores automáticos
para apoio à tomada de decisão.

## Tecnologias
- Python
- Django
- PostgreSQL
- Tesseract OCR
- HTML / CSS / JavaScript / Bootstrap

## Funcionalidades MVP
- Autenticação de usuários
- Cadastro de empresas
- Upload de documentos financeiros
- Extração de dados via OCR
- Cálculo de indicadores financeiros
- Geração de relatórios PDF
